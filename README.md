# AF-Kart AI – Product Review Sentiment Analysis

This is a web-based application that analyzes customer reviews using Machine Learning.

## 🚀 Features
- Product-wise review system (Headphones, Phone, Laptop)
- Sentiment analysis (Positive / Negative)
- Star rating system ⭐
- Graph visualization 📊

## 🛠️ Technologies Used
- Python (Flask)
- SQLite (for deployment)
- scikit-learn
- HTML, CSS
- Chart.js

## ▶️ How to Run
1. Install dependencies
2. Run app.py
3. Open http://127.0.0.1:5000
